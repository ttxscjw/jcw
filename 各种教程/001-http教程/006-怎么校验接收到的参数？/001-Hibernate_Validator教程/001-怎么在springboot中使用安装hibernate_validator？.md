# 怎么在springboot中使用安装hibernate validator？



## 说明

- 添加maven依赖即可



```xml
		<!-- hibernate-validator依赖 -->
        <dependency>
            <groupId>org.hibernate</groupId>
            <artifactId>hibernate-validator</artifactId>
            <version>5.4.1.Final</version>
        </dependency>
```
